# turnero-c
## Este proyecto es un sistema de turnos desarrollado en lenguaje C como trabajo final de una materia universitaria en 2023.
A lo largo del desarrollo se aplicaron diversos conceptos fundamentales de programación, entre ellos:

- Estructuras de control
- Funciones
- Arreglos
- Registros (structs)
- Recursión
- Memoria dinámica
- Estructuras de datos dinámicas: pilas, filas y listas
- Manejo de archivos

El objetivo del proyecto fue poner en práctica los conocimientos adquiridos durante la cursada mediante la creación de una aplicación funcional y estructurada.
